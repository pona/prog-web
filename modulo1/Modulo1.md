# Unidad 1: Javascript (parte 1)

## 🎯 Consigna y Requisito Mínimo
Esta actividad consta de cuatro ejercicios prácticos. Para cumplir con el requisito, debes elegir y resolver al menos dos (2) de los ejercicios propuestos.


| N°     | Descripción de la Tarea | Temas de JS Involucrados |
| :--- | :--- | :--- |
| Ejercicio 1 | Declarar variables de tipo string, number y boolean, e imprimir una frase con interpolación de strings (``). | Variables, Tipos de Datos, Interpolación de Strings. |
| Ejercicio 2      | Utilizar estructura de control para determinar el medio de transporte más adecuado dada una distancia en metros (reglas: pie, bicicleta, colectivo, auto, avión).	|Estructuras de Control (condicionales).      |
| Ejercicio 3      | Crear un bucle for que recorra un array de productos e imprimir un mensaje de compra para cada elemento.      | Arrays, Estructuras de Bucles (for).      |
| Ejercicio 4      | Recorrer un array de números para devolver el número mayor.      | Arrays, Estructuras de Bucles, Lógica Condicional.      |

### Se evaluará la correcta aplicación de:

- Declaración y uso de variables.
- Implementación de estructuras de control (condicionales).
- Uso de estructuras de bucles (iteraciones).
- Aplicación de interpolación de strings.

# Unidad 2: Javascript (parte 2)

## 🎯  Consigna y Requisito Mínimo
Esta actividad consta de cuatro ejercicios prácticos. Para cumplir con el requisito, debes elegir y resolver al menos dos (2) de los ejercicios propuestos.


| #     | Descripción de la Tarea | Temas de JS Involucrados |
| :--- | :--- | :--- |
| 1     | Crear una función para calcular el promedio de dos notas parciales (ejemplo: materia Historia).      | Funciones, Parámetros, Retorno de valores.      |
| 2     | Programar un contador de caracteres para un elemento textarea.      | Manipulación del DOM, Eventos (input), Funciones.      |

### Se evaluará la correcta aplicación de:

- Creación y utilización de variables e interpolación de strings.
- Implementación de funciones y manipulación del DOM.
- Utilización de métodos de array (Ej. filter, map, reduce).
- Utilización de desestructuración (destructuring).

# Unidad 4: Next (parte 2)

## 🎯 Objetivos de Aprendizaje

- Comprender y aplicar la estructura de un proyecto Next.js.
- Crear y estructurar componentes reutilizables (funcionales y estáticos).
- Aplicar estilos CSS a los componentes para lograr un diseño visual.

### Criterios de Evaluación
Se valorará la correcta implementación técnica de los conceptos, prestando especial atención a:

- Estructura de Componentes: Uso correcto de la estructura de componentes de NEXT (definición, importación, anidamiento y reutilización).
- Implementación de Next.js: Uso correcto de las convenciones del *framework* (ej. páginas, *routing*, etc.).
- Aplicación de CSS: Correcto estilizado de la aplicación.
